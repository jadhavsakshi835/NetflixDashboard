# 📊 Netflix Power BI Dashboard

## Overview
This project is a Power BI dashboard created using the Netflix dataset. It provides a simple and interactive way to explore Netflix content through different charts and KPI cards.

## Features
- Total Netflix Titles
- Total Countries
- Total Directors
- Movies vs TV Shows Distribution
- Country-wise Title Analysis
- Release Year Analysis
- World Map Visualization
- Interactive Filters

## Dashboard Insights
- Movies make up the majority of Netflix content.
- The United States has the highest number of titles.
- The dashboard shows how Netflix content is distributed across different countries.
- Users can easily compare Movies and TV Shows.

## Tools Used
- Microsoft Power BI
- Microsoft Excel (Dataset)

## Dataset
The dataset contains information about Netflix titles, including:
- Show ID
- Title
- Director
- Country
- Release Year
- Rating
- Type (Movie/TV Show)

## Purpose
The aim of this project is to practice data visualization and create an interactive dashboard that helps users understand Netflix content using Power BI.
